# LLM Model Capability Classification Project

## Overview

This project uses Machine Learning techniques to classify LLM model capabilities using:

* Logistic Regression
* Decision Tree
* GridSearchCV
* Feature Importance
* Accuracy & F1 Score Evaluation

## Technologies Used

* Python
* Pandas
* NumPy
* Scikit-learn
* Matplotlib
* Seaborn

## Features

* Data Cleaning
* Exploratory Data Analysis (EDA)
* Information Value (IV) Calculation
* Feature Selection
* Logistic Regression Model
* Decision Tree Model
* Hyperparameter Tuning using GridSearchCV
* Model Evaluation

## Dataset

The dataset contains information about Large Language Models (LLMs) and their capabilities.

## Evaluation Metrics

* Accuracy
* Precision
* Recall
* F1 Score
* Confusion Matrix

## How to Run

Install required libraries:

```bash
pip install -r requirements.txt
```

Run the project:

```bash
python main.py
```
